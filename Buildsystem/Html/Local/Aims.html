The main aims and design principles of the OKlibrary

The OKlibrary is a research platform on generalised satisfiability, providing
  • a generative C++ library
  • and extensions by computer algebra and proof assistants.
Special services provided:
  • An extensive documentation system, ranging from extracting program documentation to planning support.
  • The OKlibrary offers installation and usage support for packages and tools used for programming and related activies.
