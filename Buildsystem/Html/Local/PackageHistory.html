The package-history of the OKlibrary

Version 0.2

  1. 00081:
    • Improved packages:
      1. README updated
      2. "all" extended
      3. the new CLisp should build also on 64-bit machines
      4. new versions of Gmp, R and git
      5. boost and mhash now local by default.
    • Various extensions at the Maxima/Lisp level, especially regarding combinatorial matrices.
    • Several SAT-related plans at the Maxima/Lisp level.
  2. 00082:
    • Work on installing Maxima through ECL:
      1. Installation is now functional.
      2. Some wrong assumption of our testsystem regarding Lisp hash functions were repaired.
      3. But there are still bugs in the Maxima/ECL combination.
    • Several updates of the build system.
    • Some work on AES (at the Maxima level).
    • Planning on new SAT functionality at Maxima/Lisp level.
    • Planning on graphs, semigroups and bioinformatics at Maxima/Lisp level.
  3. 00083:
    • Maxima installation:
      1. Updated to Maxima 5.16.3.
      2. Work on removing the ECL-related bugs (with the help of the Maxima mailing list).
    • Maxima usage:
      1. Started improving the set-operations (with the help of the Maxima mailing list).
      2. And improving on apply-functionality.
    • Experimental investigations (especially Ramsey problems).
    • Some buildsystem updates.
    • More graph functionality (Maxima/Lisp level).
  4. 00084:
    • Improved Maxima installation (new installation is recommended).
    • More basic graph functionality at Maxima/Lisp level.
    • Some maintenance work on OKsolver-2002.
    • Further (initial) investigations into Ramsey problems.
    • Update to git version 1.6.0.2.
    • Several other additions at Maxima/Lisp level.
